# content safety demo

This demo code illustrates the following Azure Services:

* Content Moderation (text)
* Image Moderation (via the SDK)
* Image Moderation (via the API)

## Environment Variables

In order to run this code, you will need to set two environment variables:

- `CONTENT_SAFETY_KEY`
- `CONTENT_SAFETY_ENDPOINT`

## Sample images

This code assumes that you have an `images` directory that contains exaples of images that would trigger
content moderation. See the following "image" data sets for an example:

- See this [Graphical Violence data set on Kaggle](https://www.kaggle.com/datasets/kartikeybartwal/graphical-violence-and-safe-images-dataset/data)
- You can also download the data from [this dropbox link](https://www.dropbox.com/scl/fi/7xu3xbv8jpkracmhek3sq/archive.zip?rlkey=ow7cupw80w50h94q4sam2chsa&dl=0)
